######### To get task scheduler to run ##########
#################################################
1) Cpen task scheduler
2) Clicktask scheduler library 
3) Create task 
4) Name the process
5) Go to actions tab
6) Hit New
7) In Program/script find python executable on the machine (((C:\pathtoptyhonexecutable\python.exe))) [[to find use where python in cmd]]]
8) In add arguments(optional): input put the (((app.py or run.py))) file for the flask application 
9) In Start in (optional) Tab put the path leading up to that run.py or app.py file for flask (((C:\Users\path\tofolder\holding_app_file )))
10) Click ok close 
11) Double click on it to reopen 
12) Navicagte to triggers tab
13) Click daily and change the time you want 
14) Click ok 
15) To test out scheduler earlier click on task name and in right actions tab hit run
